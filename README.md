# Birthday_Card
**Greeting Card App**

A simple Android Greeting Card application built using Kotlin and Jetpack Compose. This project demonstrates modern Android development practices including declarative UI with Compose, Material 3 design components, and Gradle version catalogs for dependency management.

📱** Project Overview**

The Greeting Card App is a basic Android application that displays greeting messages using Jetpack Compose UI components. It serves as a beginner-friendly example for learning:

Jetpack Compose UI

Material 3 design system

Android Gradle configuration

Kotlin-based Android development

Version catalog dependency management

🚀 Features

Modern Jetpack Compose UI

Material 3 components

Kotlin-first Android development

Clean and minimal project setup

Android testing support

Debug UI tooling support

🛠️ Tech Stack

Language: Kotlin

UI Framework: Jetpack Compose

Build System: Gradle (Kotlin DSL)

Architecture: Android standard project structure

Testing: JUnit + AndroidX testing libraries

📋 Requirements

Before running the project, ensure you have:

Android Studio Hedgehog or newer

JDK 11

Android SDK 35

Gradle with Version Catalog support
